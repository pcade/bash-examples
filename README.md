# bash-examples


## **Арифметические операторы:**

- `+` — сложение
- `-` — вычитание
- `*` — умножение
- `/` — деление
- `%` — остаток от деления


## **Операторы сравнения:**

- `-eq` — равно
- `-ne` — не равно
- `-lt` — меньше
- `-le` — меньше или равно
- `-gt` — больше
- `-ge` — больше или равно

## **Логические операторы:**

- `&&` — логическое И
- `||` — логическое ИЛИ
- `!` — логическое НЕ
- `|` - пайп для перенаправления вывода из команды в команду, например grep
